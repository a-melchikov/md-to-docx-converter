# md-to-docx-converter

Веб-приложение для конвертации Markdown в DOCX с настраиваемыми стилями, предпросмотром и импортом/экспортом конфигураций.

## Возможности

* Конвертация Markdown в DOCX.
* Поддержка базовых элементов Markdown:

  * заголовки;
  * обычный текст;
  * списки;
  * таблицы;
  * цитаты;
  * inline code;
  * блоки кода;
  * ссылки;
  * изображения.
* Настройка оформления через UI:

  * шрифты;
  * размеры текста;
  * жирность, курсив, подчёркивание;
  * отступы;
  * интервалы;
  * межстрочный интервал;
  * параметры страницы;
  * стили списков и таблиц.
* Импорт и экспорт настроек в JSON.
* Предварительный просмотр результата перед выгрузкой DOCX.
* Предупреждения о неподдерживаемых элементах Markdown и неконвертируемых символах.

## Планируемая архитектура

```text
frontend/   React + TypeScript
backend/    Node.js + TypeScript + Fastify
shared/     общие типы и JSON-схемы
infra/      Docker, nginx, compose-файлы
```

## Запуск в Docker

```bash
docker compose up -d --build
```

После запуска приложение будет доступно по адресу:

```text
http://localhost
```

## Настройки конвертации

Настройки можно задавать двумя способами:

1. Через интерфейс приложения.
2. Через импорт JSON-файла конфигурации.

Пример конфигурации:

```json
{
  "document": {
    "pageSize": "A4",
    "orientation": "portrait"
  },
  "styles": {
    "heading1": {
      "font": "Times New Roman",
      "size": 16,
      "bold": true
    },
    "paragraph": {
      "font": "Times New Roman",
      "size": 12
    },
    "inlineCode": {
      "font": "Courier New",
      "size": 11
    }
  }
}
```

## Статус проекта

Проект находится на этапе проектирования и начальной реализации.

## Основная цель

Сделать удобный инструмент, который позволяет быстро преобразовывать Markdown-документы в DOCX без ручного форматирования в Microsoft Word или LibreOffice.
